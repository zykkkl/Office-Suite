# Strategic Planning

> Domain: Business plans, fundraising decks, project proposals, startup pitches, strategic roadmaps, board presentations
> Cognitive posture: Decision-framing -- the audience evaluates investment-worthiness, risk, and return potential

## Cognitive Basis

Strategic presentations must satisfy two simultaneous cognitive demands:

1. **Credibility establishment**: The audience must believe the presenter understands the landscape (market, competition, risks)
2. **Return visualization**: The audience must see a clear path from current state to desired outcome with quantified waypoints

This dual demand creates a **two-act structure**: the first half builds the case for why change is needed (problem urgency); the second half demonstrates capability to deliver change (solution confidence).

## Density Target: Dense (70-85%)

Strategic audiences (investors, executives, board members) evaluate density as a proxy for analytical depth. Each slide should carry: one strategic claim + 2-3 quantified data points + at least one chart or framework diagram.

Core data pages (market sizing, financial projections, ROI) can push toward 85% fill. Vision and summary pages should breathe at 60-70%.

## Visual-Verbal Ratio

Approximately 45% text and data, 35% charts and frameworks, 20% imagery and visual anchors.

Charts dominate market analysis and financial slides. Framework diagrams (2x2 matrices, process flows, timelines) dominate strategic positioning slides. Imagery (product screenshots, team photos) provides human connection.

## Color Derivation

Temperature: cool-authoritative with a premium accent. The palette must project strategic foresight and professional gravitas.

| Token | Default | Rationale |
|-------|---------|-----------|
| accent | Deep amber/gold or brand primary | Premium signal for key metrics and CTAs |
| bg_primary | Dark navy for key slides (market, financials); white for content slides | Dark = emphasis moments; white = analytical clarity |

Key slides (market size, traction, financial projections) may use dark backgrounds to create visual punctuation in the deck rhythm.

## Typography Posture

Every slide title is a **strategic claim**, not a topic label.

Wrong: "Market Analysis"
Right: "Global TAM $42B Growing at 18% CAGR, China Represents 35% Share"

Title wording should be decisive and specific. Vague titles ("Our Strategy", "The Opportunity") signal vague thinking.

Big numbers (market size, revenue, ROI) use the largest available font with the accent color. The number itself is the visual protagonist.

## Strategic Narrative Frameworks

| Framework | Sequence | Use When |
|-----------|----------|----------|
| **Problem-Solution-Return** | Pain point -> Solution path -> Quantified ROI | Fundraising, project proposals |
| **Trend-Position-Execution** | Market shift -> Strategic position -> Implementation plan | Board presentations, strategic reviews |
| **Current-Target-Bridge** | Where we are -> Where we're going -> How we get there | Transformation roadmaps |

Each slide maps to one node in the framework. The transition between slides creates the logical chain.

## Content Expression

| Strategic Element | Visual Encoding |
|------------------|----------------|
| Market sizing (TAM/SAM/SOM) | Nested circles or waterfall decomposition |
| Competitive positioning | 2-axis matrix with company highlighted |
| Financial projections | Combo chart (bar for revenue, line for margin) |
| Implementation roadmap | Timeline with phase markers and milestones |
| Risk assessment | Probability x Impact matrix |
| Team capability | Photo + title + one-line credential |
| Funding allocation | 4 independent donut charts (not a single combined chart) |
| Unit economics | Big number decomposition (CAC, LTV, margin) |

## Prohibited Patterns

| Pattern | Why It Fails |
|---------|-------------|
| Claims without data | Strategic audiences verify every assertion |
| Vague competitive analysis | "We have no competitors" destroys credibility |
| Returns described qualitatively | "Significant growth potential" is meaningless without a number |
| Over-decorated slides | Signals style over substance in an analytical context |
| More than 7 information points per page | Exceeds decision-maker attention budget |
| Generic stock images | Use actual product screenshots or team photos |
