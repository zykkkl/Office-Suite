# Brand and Promotion

> Domain: Product launches, brand campaigns, creative portfolios, art exhibitions, cultural events, marketing pitches
> Cognitive posture: Affective-first -- the audience responds to emotional impact before analytical content

## Cognitive Basis

Promotional presentations exploit the **affect heuristic** (Slovic, 2001): emotional impressions formed in the first seconds of viewing anchor all subsequent evaluation. The design must create a strong positive affect in the pre-attentive window (first 200ms), then sustain it through the narrative.

This inverts the standard hierarchy. In analytical presentations, content drives design. In promotional presentations, design drives engagement -- the audience decides whether to invest attention based on visual quality before reading a single word.

## Density Target: Sparse (40-55%)

Each slide delivers exactly one emotional or conceptual payload. White space is not emptiness -- it is **visual silence** that gives each element room to resonate.

The fewer elements on a slide, the more weight each element carries. A single word in 40pt type centered on a dark canvas has more impact than ten bullet points.

## Visual-Verbal Ratio

Approximately 30% text, 10% data/charts, 60% imagery and visual composition. Text is refined to slogan density -- every word is chosen for rhythmic and emotional impact, not informational completeness.

## Color Derivation

Temperature: determined by brand identity or thematic association. Promotional palettes have permission to be bold, unconventional, and emotionally specific.

| Token | Default | Rationale |
|-------|---------|-----------|
| accent | Brand primary color, or a thematic color with strong emotional association | This color IS the brand on every slide |
| bg_primary | Dark for dramatic immersion, or white for clean premium | Depends on brand personality |

Subtle gradients within the accent color family are permitted to create atmospheric depth. Cross-family gradients are prohibited.

## Typography as Visual Medium

In promotional design, typography is not a hierarchy tool -- it is a **visual medium**. Letterforms carry aesthetic weight independent of their semantic content.

| Element | Treatment |
|---------|-----------|
| Hero text (cover, key moments) | Oversized (36-40pt), generous letter spacing, centered |
| Slogan text | Refined, minimal, each word earns its place |
| Body text | Light weight, moderate size, restrained presence |
| KPI numbers | Extra-large, the number IS the visual focal point |

Weight contrast (bold vs light) creates more sophisticated hierarchy than size contrast alone.

## Slide Archetypes

| Archetype | Composition | Use |
|-----------|------------|-----|
| **Hero moment** | Single oversized element centered, generous whitespace | Cover, key message, closing |
| **Immersive image** | Full-bleed image + semi-transparent overlay + text | Scene-setting, emotion-building |
| **Data spotlight** | Extra-large number + minimal annotation | KPI, milestone, achievement |
| **Split composition** | Asymmetric division (e.g., 30:70) between image and text | Product showcase, feature highlight |
| **Atmospheric** | Solid/gradient background + single line of text | Transitions, breathing moments |

No fixed title bar. No page numbers. No navigation elements. Each slide is a self-contained visual experience.

## Prohibited Patterns

| Pattern | Why It Fails |
|---------|-------------|
| Dense text paragraphs | Destroys the sparse, premium atmosphere |
| Complex data tables | Wrong medium for affective communication |
| Generic stock photography | Undermines brand authenticity |
| Corporate template aesthetics | Signals "default" rather than "designed" |
| Information cramming | Each slide needs only one message |
| Small font sizes and tight spacing | Oppressive rather than premium |
